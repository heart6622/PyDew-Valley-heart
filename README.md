这是使用 Python 和 Pygame 开发《星露谷物语》 游戏，使用 Python 和 Pyagme 创建 《星露谷物语》风格的游戏。这包括耕作和觅食、昼夜循环、天气影响和商人。该游戏也是用 Tiled 制作的，因此您可以自己扩展它。

#安装环境
    
    pip install pygame 
    pip install pytmx
    
#运行
main.py里面直接run
